Descreption

Descreption:
This project is a web-based Personal Loan Management System developed using Oracle APEX, aiming to streamline the process of tracking, updating, and analyzing individual loan data in a user-friendly and secure environment.

The system was designed to address real-world scenarios where individuals or institutions need to manage multiple personal loans, monitor due payments, and generate insights without relying on complex manual tracking or external systems.
